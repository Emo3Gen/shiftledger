# Гайд по стилю общения — Детский центр «Про Дети»
## Автоматически извлечён из 20524 реальных QA-пар

## Общий стиль
- Обращение: на **Вы**
- Средняя длина ответа: **137 символов** (~2.6 предложений)
- Эмодзи: **1.09** на сообщение

## Типичные приветствия
- `[ИМЯ], добрый день!` (1680 раз)
- `[ИМЯ], здравствуйте!` (1634 раз)
- `[ИМЯ], добрый день! ☀` (441 раз)
- `[ИМЯ], добрый вечер!` (331 раз)
- `[ИМЯ], доброе утро!` (316 раз)

## Используемые эмодзи
✅(4538) 🌷(3287) 🌸(2281) ☀(2086) 😔(1627) ️(1262) 🌿(862) 🌼(578) ☺(576) 😌(522)

## Заметки по стилю
- Операторы используют формальное обращение на 'Вы'
- Активное использование эмодзи (в среднем 1.1 на сообщение)
- Развёрнутые ответы (в среднем 137 символов)
